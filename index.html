// 功能特性

专为通联设计的
完整工具链

从快速录入到云端同步，覆盖通联日志的每一个环节。

USB LSB FM CW FT8
📻

通联日志

按日期分组卡片式展示，模式彩色标签，左右滑动快速删除，呼号自动联想与大写转换

📝

智能录入

自定义 RST 数字键盘、功率键盘、快捷标签，支持一次性输入自动解析填充

📍

位置与网格

自动获取 GPS 位置并计算梅登海德网格坐标，B 台分区自动识别省份

📤

ADIF 导入导出

标准 ADIF 格式支持，自动解析日期创建对应通联记录

☁️

Cloudlog 同步

连接 Cloudlog 站台，ADIF 格式上传 QSO，自动上传开关，同步结果详情

📡

设备天馈管理

天线品牌型号分组管理，长按拖拽排序，左右滑动删除

🌙

深色模式

完整 Material 3 深色主题，WindowSizeClass 自适应屏幕尺寸

⏱️

时区与设置

默认 UTC+8，可切换时区，OP 信息配置，检查 GitHub Release 更新
